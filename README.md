#### Instructions to execute the script for wpengine-interview

- Install pre-requisites
  - Python3 [1]
  - pip [2]
- Download and extract the zip file
- Execute bash script
  ```
  bash wpmerge input.xlsx output.csv
  cat output.csv
  ```
